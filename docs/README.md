# MarketBook 项目文档

## 📁 文档结构

```
docs/
├── README.md              # 本索引
├── 路线图.md              # 短期/中期/长期目标
├── PROJECT_STRUCTURE.md   # 项目目录结构（以实际为准）
├── marketbook项目结构.md  # 项目规划与模块愿景
├── archive/               # 历史归档
│   ├── design/          # 设计/UI 优化报告
│   ├── seo/             # SEO 优化报告
│   ├── reports/         # 功能开发总结
│   ├── html/            # 历史首页版本
│   ├── config/          # 历史 sitemap、robots 等
│   └── other-projects/  # 其他项目（RadarAI）
```

## 📌 统领性文档

| 文档 | 位置 | 说明 |
|------|------|------|
| 项目总览 | 根目录 README.md | 快速开始、状态、文档索引 |
| 项目现状 | 根目录 项目现状说明.md | 完成情况、待办 |
| 路线图 | docs/路线图.md | 短期/中期/长期目标 |
| 开发 SOP | SOP/ | 平台功能、模拟盘开发流程 |
| 项目结构 | docs/PROJECT_STRUCTURE.md | 源码目录说明 |

## 📦 归档说明

- **design/**：设计优化、微交互、可访问性等 UI 相关报告
- **seo/**：SEO 优化相关文档
- **reports/**：功能开发总结、模拟盘/用户系统等优化报告
- **other-projects/**：非 MarketBook 项目（如 RadarAI 分析）

需要查找历史优化记录时，到对应子目录即可。
