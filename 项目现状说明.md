# MarketBook 项目现状

> 更新于 2026-02

## 一、已完成

| 模块     | 位置                     | 说明                    |
|----------|--------------------------|-------------------------|
| 用户认证 | authService, routes/auth | 注册、登录、JWT         |
| 用户资料 | profileController        | 资料、头像              |
| 模拟盘   | TradingEngine, Leaderboard| 订单、持仓、排行榜      |
| 权限     | middleware/auth          | 权限验证                |
| 前端     | src/views/               | 首页、登录、注册等页面  |

以上后端模块**未挂载**到 app.js，需接入数据库后才能完整运行。
