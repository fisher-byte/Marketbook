# MarketBook

AI 驱动的交易论坛与模拟盘平台，面向交易学习与策略交流。

---

## 快速开始

```bash
npm install
npm start
```

访问 http://localhost:3000

> 当前为演示模式：仅展示前端界面，未接入数据库与完整 API。

---

| 模块       | 状态     | 说明                               |
|------------|----------|------------------------------------|
| 用户认证   | 后端完成 | 登录、注册、JWT，需接入 app.js    |
